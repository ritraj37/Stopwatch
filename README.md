# ⏱️ Stopwatch Web App

A simple **Stopwatch application** that allows users to measure elapsed time. The stopwatch includes **start, pause, and reset** functionalities and updates time in real-time.

This project demonstrates **JavaScript timing functions, DOM manipulation, and event handling**.

---

## 🚀 Features

* Start the stopwatch
* Pause the stopwatch
* Reset the stopwatch
* Real-time time display
* Clean and responsive UI
* Beginner-friendly JavaScript project

---

## 🛠️ Technologies Used

* **HTML5** – Structure of the application
* **CSS3** – Styling and layout
* **JavaScript** – Time calculation and functionality

## ▶️ How to Run the Project

1. Download or clone this repository
2. Open the project folder
3. Run **index.html** in your web browser
4. Use the buttons to start, pause, and reset the stopwatch

---

## 📌 Project Purpose

This project was created to practice **JavaScript timing functions like `setInterval()` and DOM manipulation** while building a useful time-tracking tool.
